# Simple Markdown Example

This is a basic example showing common Markdown features.

## Text Formatting

You can make text **bold** or *italic*. You can also use __bold__ and _italic_.

## Lists

Unordered list:
- First item
- Second item
- Third item
  - Nested item
  - Another nested item

Ordered list:
1. First step
2. Second step
3. Third step

## Links

Check out [Nomadnet on GitHub](https://github.com/markqvist/nomadnet) for more info.

## Code

Inline code: `print("Hello, World!")`

Code block:
```
def greet(name):
    return f"Hello, {name}!"
```
